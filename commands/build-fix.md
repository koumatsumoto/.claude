# ビルド修正

TypeScript とビルドエラーを段階的に修正:

1. ビルドを実行: npm run build または pnpm build

2. エラー出力を解析:
   - ファイルごとに整理
   - 重大度でソート

3. 各エラーについて:
   - エラー文脈を表示（前後 5 行）
   - 問題を説明
   - 修正案を提示
   - 修正を適用
   - ビルドを再実行
   - 解決を確認

4. 中止条件:
   - 修正で新たなエラーが出た
   - 同じエラーが 3 回続く
   - ユーザーが一時停止を要求

5. サマリを表示:
   - 修正済みエラー
   - 残っているエラー
   - 新規発生エラー

安全のため 1 回に 1 エラーずつ修正すること！
